# Dice Roller

This is a small game called dice roller cerated using Python programming language. Random module has been included to create the main function of this program and GUI was created using tkinter. 
All images of dice including favicon were created using GIMP. 

## Usage Instruction

* First guess a number from 1 to 6.
* Type the guess number inside the entry box.
* After that press roll button and see the result below the answer label.
* For try agin, please use retry button.
