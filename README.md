# Dice Roller

This is a small game called dice roller cerated using Python programming language. Random module has been included to create the main function of this program and GUI was created using tkinter. 
All images of dice including favicon were created using GIMP. 

## Usage Instruction

* First of all you have to guess a number from 1 to 6.
* You can type the guess number inside the entry box.
* After that you can press roll button and see the result below the answer label.
* If you want to try agin, please use retry button.
